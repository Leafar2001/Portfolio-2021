The project

Medifit is a local gym that were in need of a website overhaul & update. I was able to do this for them, using a CMS such as WordPress because they wanted to be able to edit content from time to time. The goal of the new website was to give it a more modern look & feel, and to be able to add modern systems like news articles, mobile support, and online registration to the site.

Updating Medifit's website started off as a school project. I was handed the assignment to update the branding of an either fictional or existing company. Having known the owners of Medifit for a while, I knew they could do with a new website, so I proposed the idea to them and showed them a simple mock-up of what i had in mind. They liked the idea and were excited to work with me.

Because of how old the website was, i decided not to update the existing infrastructure but to fully rebuild it by setting up a new database and CMS, in this case WordPress. I chose WordPress because of its popularity and plugin support, so whenever Medifit needed something changed, I was able to do this quickly, or able to let them do it by themselves.

Design

For the design of this website, I chose a modern business color theme while still holding on to the existing branding of Medifit. This would ultimately lead to a website using sans-serif fonts and a primary color of blue. I used the primary color to highlight important elements such as call to action buttons.

Development

I developed the site using a local copy of WordPress, developed the key features and implemented the Medifit branding, and then showed it to Medifit. They were ecstatic with the result and really liked the new and improved look. I tried keeping the Medifit branding in mind by staying with the same color scheme they use throughout their buildings and other graphics. I'm happy that i was able to transform this website from the stone-age to a modern and competitive website that put Medifit on the map.

Results

The only deliverable for this project is the website, as there weren't many other digital Medifit assets that needed modernising.

Website

By building this website in a CMS, in this case WordPress, I was able to get familiar again with WordPress after not having used it for a while. It was good to refresh my CMS and database knowledge, as well as how to be involved with real clients and businesses.
