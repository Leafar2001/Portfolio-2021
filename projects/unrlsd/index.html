2022 • Branding • UX / UI

Creating an opportunity for fans to remix their favourite artists' music.

Unrlsd is an online music platform where artists can share unreleased music and snippets to their fans. Fans can interact with them by remixing these snippets and this way the artist can build hype around upcoming releases, aswell as seeing if their fanbase would enjoy their next release.

This project was built in a short period of 3~ weeks as an individual school project, with the end result being a clickable Adobe XD prototype of the UI design.

The project

The assignment was simple: find a topic of interest and build something digital that users can interact with. I was pretty free in what type of prototype I wanted to build, so I opted for something I wanted to learn more about, which is building web apps with the React framework.

To document my vision for this project, I created a project plan:



Projectplan

Since starting the project I figured out I won't have the time to create a fully functional web app so I will instead be focusing on building clickable prototype built with XD, unlike the live web app prototype listed in the project plan.

Research

In the document linked below you can find my general research findings for the project. The goal for this research period was to figure out how I wanted to approach the project, what concept I ended up working on and how I would build it all.

The research will consist of mostly library research using the DOT framework and is based on the following research questions:

- How can an artist create a fun interaction with their fans to create new music?

- What are effective ways to involve fans in the studio process?

- What makes for a smooth cooperation between fan and artist?

Branding

The goal for the branding proces was to create a unique and vibrant style, and serve as a design asset for myself to base designs off of.

I started by looking at color palettes and ended up settling for a vibrant color palette with complementing darker tones. I chose this palette because the music industry is often vibrant so I didn't want any pastels in the branding of Unrlsd.

Unrlsd - Colors

Although music has a global target audience, social media and being able to remix something would be seen as a younger target audience. Therefore I wanted to use modern fonts that would compliment eachother well. I chose Gilroy for clean and bold headings, and Poppins for sharp but playful paragraphs.

Unrlsd - Fonts

With the color and fonts set, I was able to start messing around in Adobe Illustrator for logo concepts. I looked at different color schemes, icons and symbols that would combine well with the primary font.

Unrlsd - Logo concepts

The end result of the logo consists of the primary gradient on the letter U (for Unrlsd) with the right part of the U being cut off, combining with the 2 dots to symbolize a music note.

Unrlsd - Logo result

Design

I designed the UI in Adobe XD. My goal for this design was to create a modern UI that had all the elements I needed to build into a prototype and start testing. The focus for version 1 was to create all the important screens such as the home feed, the remix functionality, artist profiles and general navigation.

Once I was satisfied with version 1 and got some feedback on it, I started converting the design into a prototype to be able to run tests on user flow and UX.

Unrlsd - UI design v1

After the prototyping phase, I made changes based on user feedback and conclusions that I took from my usability tests and ended up with version 2, the finished version, of the design.

Unrlsd - UI design v2

Prototyping

The prototyping phase was done not only to come up with a dynamic and interactive deliverable, but also to test the design. The prototype was made by using the built-in prototyping functionalities of Adobe XD on the UI design. I created links for all the important functionalities I wanted to test.

With the prototype done, I was ready to choose a test method. Since I wanted to test user flow and design choices, I decided to use Maze, the online prototype testing tool. This tool has seemless integration with Adobe XD and allows me to have users test the usability of my prototype and then answer questions afterwards. Maze also provides me with success rates, misclicks, and heatmaps for clicks that the users went through.

After testing, I was able to come to a couple conclusions that allowed me to make positive changes to the design. Everything is documented in the file linked below.
