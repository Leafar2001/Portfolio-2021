Creating an online platform for artists to test new material out.

Unrlsd is an online music platform where artists can share unreleased music and snippets to their fans. Fans can interact with them and this way the artist can build hype around upcoming releases, aswell as seeing if their fanbase would enjoy their next release.

This project was built in a short period of 3~ weeks as an individual school project. Since it was developed locally, there is no live demo available.

The project

The assignment was simple: find a topic of interest and build something digital that users can interact with. I was pretty free in what type of prototype I wanted to build, so I opted for something I wanted to learn more about, which is building web apps with the React framework.
